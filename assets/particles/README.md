# Particles

Efek partikel (bintang berkelip, energi spiritual mengambang, dan ledakan aura)
dibuat secara **prosedural** melalui `<canvas>` dan CSS di dalam `script.js` /
`style.css`, sehingga tidak memerlukan file gambar.

Jika ingin memakai tekstur partikel kustom (mis. `spark.png`, `glow.png`),
letakkan di folder ini lalu rujuk dari CSS/JS sesuai kebutuhan.
